# Action Übung 1:  BGR zu HSV

In dieser Übung wird der HSV Farbraum betrachtet. Die Information der Farbe wird
durch die drei Werte  

 - H: Helligkeit (Hue)
 - S: Sättigung (Saturation)
 - V: Value (Helligkeit)
 
repräsentiert.
